Ideally, this page should contain a complete list of all items owned by the club (except for temporary and consumable items such as stash to be sold and food). It is currently being compiled for the first time. All new gear aquired should be added to this list, and any retired or damaged gear should be removed from it as soon as the item leaves the tackle store.
